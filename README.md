# UBLC-AI-Automation-TeamLibby

## Project Summary

Libby is an AI-powered library assistant chatbot developed for the University of
Batangas Lipa Campus (UBLC). The system automates common library-related inquiries
such as book searching, library hours, borrowing rules, and discussion room
availability through a simple conversational web interface.

The goal of this project is to demonstrate how AI and automation can improve access
to institutional services while following proper DevOps practices using GitHub.

## Features and Technologies Used

### Key Features

- AI-powered chatbot for library inquiries
- Book search and basic library information access
- Automated responses for borrowing rules and policies
- Discussion room inquiry support
- Responsive web-based interface

### Technologies Used

Frontend: React, TypeScript, Vite, Tailwind CSS

Backend: AI and automation workflows (planned / simulated)

Version Control: Git & GitHub

CI/CD: GitHub Actions (Super Linter)

Data Storage: JSON / Local datasets

## Setup and Deployment Instructions

### Prerequisites

- Node.js (v18 or higher)
- Git
- npm or yarn

### Deployment

The frontend can be deployed using:

- GitHub Pages
- Netlify
- Vercel

## CI/CD Process Details

This project uses GitHub Actions for Continuous Integration.

### CI/CD Workflow

**Triggered When:**

- Pushing to the main branch
- Creating a pull request

### Pipeline Steps

1. Checkout repository
2. Run GitHub Super Linter
3. Validate code quality and formatting

### Purpose

- Ensures clean and consistent code
- Detects errors early
- Follows DevOps best practices
