# UBLC-AI-Automation-TeamLibby

# Project Summary

Libby is an AI-powered library assistant chatbot developed for the University of Batangas Lipa Campus (UBLC). The system automates common library-related inquiries such as book searching, library hours, borrowing rules, and discussion room availability through a simple conversational web interface.

The goal of this project is to demonstrate how AI and automation can improve access to institutional services while following proper DevOps practices using GitHub.
